Project information

Develop a cognitive robotic system (CRS) that includes:

  • Motion decision making: Soar cognitive agent
  • Local path planner: Modified dynamic window approach
  • Project date: August 2020 ~ July 2021
  • Project Video: https://youtu.be/32laLi58XZ4

Descriptions

Given

  • A single mobile robot (Omorobot R1).
  • Ultra-wideband (UWB) sensors.
  • A Realsense camera.
  • (A map of the environment is not given).

    • Task

    A robot autonomously navigates to a moving target

  • A human carrying an UWB sensor walks in front of the robot.
  • The robot has to detect and moves toward the target human.
  • The robot has to detect random obstacles and avoids them.

    • Experimental environment: corridor

  • OS: Ubuntu 18.04.
  • Middleware and software: ROS Melodic, Soar 9.6.0.
  • Programming language: Python (version 3.x), Soar markup language.
